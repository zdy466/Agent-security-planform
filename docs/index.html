🚀 快速开始

仅需几行代码，即可为您的 AI Agent 添加全面的安全保护

# 安装 pip install git+https://github.com/zdy466/Agent-security-planform.git # 基础使用 from agentshield import AgentShieldClient client = AgentShieldClient() result = client.process_input("用户邮箱: test@example.com") # 数据加密 from agentshield.security import DataEncryptor encryptor = DataEncryptor({"algorithm": "fernet"}) encrypted = encryptor.encrypt({"ssn": "123-45-6789"}) # IP限流 from agentshield.security import RateLimiter rate_limiter = RateLimiter({"default_rate": 100}) result = rate_limiter.check_rate_limit("192.168.1.1") # LLM 网关 from agentshield.security import LLMGateway gateway = LLMGateway({"endpoints": [...]}) response = gateway.chat([{"role": "user", "content": "Hello"}])
